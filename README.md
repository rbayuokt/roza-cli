# Roza CLI

Roza CLI is a clean, minimal terminal app for Ramadan and prayer schedules.

## Features
- Automatic first-run setup with location detection
- Daily prayer schedule with a clean, readable table
- Highlights current and upcoming prayers
- Works with the Aladhan API

## Getting Started

```sh
npm install
npm run build
node dist/cli.js schedule
```

## Dev Shortcut

```sh
npm run dev:schedule
```

## Reset Config

```sh
node dist/cli.js reset
```
